Project Background

Model Card Data Converter

Thsi video demostrates the usage of a Rains Model Card Data Converter which produces accountability trace from data captured by Google's Model Card Toolkit. The example is based on the official Model Card toolkit example

  • 0:00 - 1:56 An overview of the code for training Keras DNN classifier (part of the original example)
  • 1:56 - 2:46 An overview of the code for producing Model Card (part of the original example)
  • 2:46 - 3:53 An overview of the code for producing accountability trace from Model Cards with a failed constraint check (Rains contribution)
  • 3:53 - 5:18 A process of adding the information about license to the model card and re-creating the accountability trace (Rains contribution)
