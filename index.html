    • The fabric captures knowledge graph described using EP-Plan, PROV-O, SAO, and RAINS. Here, PROV-O provides the base vocabulary for describing accountability traces, EP-Plan extends PROV-O with mechanism for representing plans, SAO extends EP-Plan with core concepts to describe accountability plans and accountability traces, and finally, RAINS extends SAO with additional subclasses of sao:AccountableAction, sao:AccountableResult and sao:InformationElement and object and data properties to describe AI systems with ML components.
